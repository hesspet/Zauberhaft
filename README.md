# Zauberhaft

Startseite und Verteiler für Peters verrückte Ideen — eine statische Single-Page-Website, publiziert über GitHub Pages.

## Lokale Vorschau

Jeden modernen Browser auf `index.html` zeigen, oder:

```powershell
start index.html
```

Kein Build-Schritt, keine Abhängigkeiten — reines HTML5 + CSS.

## Veröffentlichung auf GitHub Pages

1. Repository auf GitHub anlegen: `hesspet/Zauberhaft`
2. In den Repository-Settings unter **Pages** den `main`-Branch als Source auswählen
3. Die Seite ist unter `https://hesspet.github.io/Zauberhaft/` erreichbar
4. Optional: Custom Domain in den Pages-Settings konfigurieren
